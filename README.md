# Реалтермо — заявки с сайта

Сайт отправляет заявки из формы в `POST /api/lead`, а сервер пересылает их на почту.

## Что настроить

Создайте `.env.local` на основе `.env.local.example`.

Обязательные переменные:

- `SMTP_HOST`
- `SMTP_PORT`
- `SMTP_USER`
- `SMTP_PASSWORD`
- `LEADS_EMAIL_TO`

Для текущего сценария письма уходят на `rtermo@yandex.ru`.

## Как это работает

1. Посетитель заполняет форму на сайте.
2. Сервер валидирует данные и защищает форму от спама.
3. Сервер отправляет письмо с заявкой на указанный адрес.

## Проверка

1. Заполнить `.env.local`.
2. Запустить `npm run dev`.
3. Отправить тестовую заявку с сайта.
4. Убедиться, что письмо пришло на нужный ящик.

## Юридические страницы

В проекте есть страницы:

- `/privacy`
- `/cookies`
- `/consent`

Перед публикацией заполните реквизиты оператора и сведения о почтовом сервисе в публичных переменных окружения.
